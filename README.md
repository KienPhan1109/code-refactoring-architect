# Senior Code Refactoring & Optimization Architect

This agent skill is designed to analyze, optimize, and refactor source code to meet the highest industry standards (Clean Code / High Performance) while absolutely preserving 100% of original business logic and system output behavior.

## 🚀 Key Features

- **Zero Regression Principle**: Restructuring code without altering functional behavior or API interfaces.
- **SOLID & Clean Code Practices**: Encouraging DRY, KISS, single responsibility, and decoupled architectures.
- **Complexity Optimization**: Improving time and space complexities (Big-O notation) through efficient data structures and algorithms.
- **Type Safety**: Applying strict type systems and modular designs.
- **Defensive Guarding**: Handling null/undefined inputs, empty states, and standard exception cases gracefully.

## 🔄 Execution Workflow

Every refactoring request follows a strict **4-step process**:

1. **Diagnosis — Code Smell Detection**: Categorize and detail potential issues (Critical 🔴, Major 🟠, Minor 🟡, Style 🔵).
2. **Refactoring Blueprint — Improvement Plan**: Prioritize architectural changes, performance optimizations, and style fixes.
3. **Refactored Implementation**: Rewrite the code completely without placeholders, integrating clear REFACTOR comments explaining the rationale.
4. **Before/After Impact Comparison**: Contrast metrics like complexity, type safety, readability, and document structural trade-offs.
